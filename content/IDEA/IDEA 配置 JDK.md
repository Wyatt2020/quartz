### 为当前项目配置 JDK

点击 `File` -> `Project Structrue` -> `SDKs` -> `+` -> `Add JDK …` ，开始添加新的 `JDK`

### 全局配置 [[JDK 安装|JDK]]

点击 `File` -> `New Projects Setup` -> `Structure` -> `Project` 下的 SDK 选项，将其更改为你最常用的 JDK 版本
