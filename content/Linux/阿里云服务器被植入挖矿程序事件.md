昨天服务器磁盘爆满，我怀疑是被黑客植入挖矿程序了。经过一番排查后发现，其实是有一个 Redis 容器一直连不上主库，产生大量日志，而 docker 默认没对容器日志大小做限制，导致磁盘被日志占满了，并不是服务器被黑了。结果晚上服务器真的被植入挖矿程序了，我真是带预言家（捂脸）

1. Redis 漏洞
2. docker 开启远程
